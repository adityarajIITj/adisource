Why Vector Space and Linear Independence?

Core Motivation

Since our ultimate goal is to fully characterize the behavior of linear systems of equations in terms of the existence and uniqueness of solutions, we must introduce fundamental linear algebraic concepts such as vector space.

Key Concepts

  • Linear Independence - Vectors that cannot be expressed as combinations of others
  • Matrix Rank - Number of linearly independent rows/columns
  • Vector Space - Set with defined addition and scalar multiplication

Connection to Gaussian Elimination

These concepts are intimately linked with the Gaussian elimination method, which is an algorithmic approach for:

  • Determining linear independence
  • Finding existence of solutions
  • Establishing uniqueness of solutions

Hinglish Summary

Vector Space Axioms

Definition

A vector space refers to objects for which addition and scalar multiplication are defined. These operations obey specific axioms.

Note: An axiom is a statement that is taken to be true without requiring formal proof.

Addition Properties

1

Associativity

(u + v) + w = u + (v + w)

2

Commutativity

u + v = v + u

3

Additive Identity

v + 0 = v

4

Additive Inverse

v + (-v) = 0

Scalar Multiplication Properties

5

Multiplicative Identity

1 · v = v

6

Associativity with Scalars

(αβ) · v = α · (β · v)

7

Distributive (Scalar)

α · (u + v) = α · u + α · v

8

Distributive (Vector)

(α + β) · v = α · v + β · v

Hinglish Summary

Vector space ke 8 axioms hai jo addition aur scalar multiplication ke rules define karte hai. Ye axioms ensure karte hai ki vector operations predictable aur consistent rahe. Jaise addition commutative hai (u+v = v+u), aur scalar multiplication distributive hai. Ye sab properties normal arithmetic jaisi hai lekin vectors ke liye specially defined hai.

Vector Space Examples

Example 1: Vector Addition in ℝ²

Let's consider two column vectors:

A = [3, -2]ᵀ

B = [4, 8]ᵀ

Vector addition:

A ⊕ B = [3+4, -2+8]ᵀ = [7, 6]ᵀ

Key Point: The addition preserves the dimension (2×1) and structure of the vector space.

Example 2: Scalar Multiplication

Given vector C = [7, 6]ᵀ and scalar α = 6:

α ⊙ C = 6 ⊙ [7, 6]ᵀ

= [6×7, 6×6]ᵀ = [42, 36]ᵀ

Important: This is NOT ordinary multiplication - it's a special vector space operation that preserves structure!

Growing Vector Spaces

Vector spaces are tied to vector addition operations. For ordered pairs (x,y) in ℝ²:

(x₁, y₁) ⊕ (x₂, y₂) = (x₁ + x₂, y₁ + y₂)

This creates a new coordinate system from the combination of two existing coordinate systems, preserving the geometric structure while performing algebraic operations.

Basis and Span

Basis Definition

A linearly independent set in V consisting of the maximum possible number of vectors in V is called a basis for V.

Key Point: If we add one or more vectors to a basis set, the set becomes linearly dependent.

Span Definition

The span of a set of vectors is the set of all linear combinations of those vectors with the same number of components.

Mathematical notation: span{v₁, v₂, ..., vₙ} = {c₁v₁ + c₂v₂ + ... + cₙvₙ | cᵢ ∈ ℝ}

"In the Span" Question

A frequent question in linear algebra: "Is vector w in the span of set S?"

Answer: Vector w is in the span of S if w can be exactly created by some linear combination of vectors in set S.

Example: Testing Span Membership

Vector v

[1]

[2]

[0]

Vector w

[3]

[2]

[1]

Set S

s₁ = [1, 1, 0]ᵀ

s₂ = [7, 0, 0]ᵀ

✓ Vector v IS in span(S)

We can write: v = c₁s₁ + c₂s₂

[1] = c₁[1] + c₂[7]

[2] [1] [0]

[0] [0] [0]

Solution exists: c₁ = 2, c₂ = -1/7

✗ Vector w is NOT in span(S)

The third element in w is non-zero (1), but all vectors in S have third element = 0.

Reason: No linear combination of zeros can produce a non-zero value.

Geometric Interpretation

In our example:

  • Span of S: A 2D plane embedded in 3D space
  • Vector v: A line lying within that plane (coplanar)
  • Vector w: "Pops out" of the plane (not coplanar)

Visualization: Think of a flat piece of paper (the span) in a room (3D space). Vector v can be drawn on the paper, but vector w points off the paper into the room.

Basis ek maximum set hai linearly independent vectors ka jo puri vector space ko span kar sakta hai. Span matlab hai ki kya hum kisi vector ko dusre vectors ke linear combination se bana sakte hai. Agar third element zero hai set mein to hum non-zero third element wala vector nahi bana sakte. Geometric interpretation mein span ek plane hoti hai 3D space mein.

Subspaces of Vector Spaces

Subspace Definition

Let V be a vector space. A subset W of V is called a subspace of V if it satisfies three properties:

🎯

1. Zero Vector

The zero vector of V is also in W

➕

2. Closure Under Addition

If u, v ∈ W, then u + v ∈ W

✖️

3. Closure Under Scalar Multiplication

If u ∈ W and α is scalar, then αu ∈ W

Example 1: Line Through Origin (Valid Subspace)

Let W be the graph of function f(x) = 2x, or equivalently y = 2x:

W = {(x, y) ∈ ℝ² | y = 2x}

Question: Is W a subspace of V = ℝ²?

✓ Zero Vector Check

When x = 0: y = 2(0) = 0

So (0,0) ∈ W ✓

✓ Addition Check

If u = (a, 2a) and v = (b, 2b):

u + v = (a+b, 2a+2b) = (a+b, 2(a+b)) ∈ W ✓

✓ Scalar Multiplication Check

If u = (a, 2a) and α is scalar:

αu = (αa, 2αa) = (αa, 2(αa)) ∈ W ✓

Conclusion: W is a subspace of ℝ²

Example 2: Unit Disc (Not a Vector Space)

Let V be the unit disc in ℝ²:

V = {(x, y) ∈ ℝ² | x² + y² ≤ 1}

Question: Is V a vector space?

✗ Fails Scalar Multiplication

Take u = (1, 0) ∈ V (since 1² + 0² = 1 ≤ 1)

Let α = 2, then αu = 2(1, 0) = (2, 0)

But (2, 0) ∉ V since 2² + 0² = 4 > 1

Therefore, the unit disc is NOT a vector space.

Practice Problem 3

Is V a vector space where V is the graph of f(x) = x²?

V = {(x, y) ∈ ℝ² | y = x²}
Click for Solution

Solution: V is NOT a vector space

Counterexample for scalar multiplication:

Take u = (1, 1) ∈ V (since 1² = 1)

Let α = 2, then αu = (2, 2)

But (2, 2) ∉ V since 2² = 4 ≠ 2

Counterexample for addition:

Take u = (1, 1) and v = (2, 4) both in V

u + v = (3, 5), but 3² = 9 ≠ 5

So (3, 5) ∉ V

General Subspace Properties

• The vector space ℝⁿ has dimension n and can be spanned by n basis vectors

• Standard basis for ℝ² is {(1,0), (0,1)}

• Standard basis for ℝⁿ is {e₁, e₂, ..., eₙ} where eᵢ has 1 in position i and 0 elsewhere

• For matrix A: row space = span of row vectors, column space = span of column vectors

Hinglish Summary

Subspace ek subset hai vector space ka jo teeno properties satisfy karta hai - zero vector present hai, addition ke under closed hai, aur scalar multiplication ke under closed hai. Straight line through origin ek valid subspace hai lekin circle ya parabola nahi hai kyunki ye scalar multiplication ke under closed nahi hai. Har subspace mein zero vector hona zaruri hai.

Question 1: Vector Space Verification

Verify whether the set of all 3×3 symmetric matrices forms a vector space under standard matrix addition and scalar multiplication.

Show Solution

Solution: Yes, it forms a vector space

1. Closure under addition: Sum of symmetric matrices is symmetric

2. Closure under scalar multiplication: Scalar multiple of symmetric matrix is symmetric

3. Zero matrix is symmetric: 0ᵀ = 0

4. All other axioms inherited from ℝ³ˣ³

Question 2: Span Determination

Determine if vector v = [2, -1, 3]ᵀ is in the span of S = {[1, 0, 1]ᵀ, [0, 1, 1]ᵀ, [1, 1, 2]ᵀ}.

Show Solution

Solution: Yes, v ∈ span(S)

We need to solve: c₁[1,0,1]ᵀ + c₂[0,1,1]ᵀ + c₃[1,1,2]ᵀ = [2,-1,3]ᵀ

This gives us the system:

c₁ + c₃ = 2

c₂ + c₃ = -1

c₁ + c₂ + 2c₃ = 3

Solution: c₁ = 3, c₂ = -2, c₃ = -1

Verification: 3[1,0,1]ᵀ + (-2)[0,1,1]ᵀ + (-1)[1,1,2]ᵀ = [2,-1,3]ᵀ ✓

Question 3: Subspace Properties

Let W = {(x, y, z) ∈ ℝ³ | x + 2y - z = 0}. Prove that W is a subspace of ℝ³.

Show Solution

Solution: W is a subspace

1. Zero vector: (0,0,0) satisfies 0 + 2(0) - 0 = 0 ✓

2. Closure under addition:

If u = (x₁,y₁,z₁) and v = (x₂,y₂,z₂) are in W, then:

x₁ + 2y₁ - z₁ = 0 and x₂ + 2y₂ - z₂ = 0

For u + v = (x₁+x₂, y₁+y₂, z₁+z₂):

(x₁+x₂) + 2(y₁+y₂) - (z₁+z₂) = (x₁+2y₁-z₁) + (x₂+2y₂-z₂) = 0 + 0 = 0 ✓

3. Closure under scalar multiplication:

If u = (x,y,z) ∈ W and α ∈ ℝ, then αu = (αx, αy, αz)

αx + 2(αy) - αz = α(x + 2y - z) = α(0) = 0 ✓

Question 4: Linear Independence

Determine if the vectors v₁ = [1,2,1]ᵀ, v₂ = [2,1,3]ᵀ, v₃ = [1,-1,2]ᵀ are linearly independent.

Show Solution

Solution: The vectors are linearly independent

We solve: c₁v₁ + c₂v₂ + c₃v₃ = 0

c₁ + 2c₂ + c₃ = 0

2c₁ + c₂ - c₃ = 0

c₁ + 3c₂ + 2c₃ = 0

Using Gaussian elimination, the only solution is c₁ = c₂ = c₃ = 0

Therefore, the vectors are linearly independent.

Key Takeaways

Vector Space Fundamentals

  • • A mathematical structure with defined addition and scalar multiplication
  • • Must satisfy 8 axioms ensuring consistent behavior
  • • Generalizes Euclidean vectors to abstract mathematical objects
  • • Foundation for linear algebra and transformations

Basis and Span

  • • Basis: Maximum linearly independent set
  • • Span: All possible linear combinations
  • • Basis vectors can generate the entire vector space
  • • Adding vectors to basis creates linear dependence

Subspaces

  • • Subset of vector space with same structure
  • • Must contain zero vector
  • • Closed under addition and scalar multiplication
  • • Lines through origin are subspaces; circles are not

Applications

  • • Essential for data science and machine learning
  • • Used in computer graphics and animations
  • • Foundation for solving linear equation systems
  • • Critical for understanding transformations
